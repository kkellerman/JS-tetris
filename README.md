# JS-Tetris
Modeled after a Udemy course, this webapp uses plain vanially JS to create a Tetris like Game


## In this game of Tetris the following native JavaScript methods:

* addEventListener()
* querySelector()
* querySelectorAll()
* keyCode()
* Math.floor()
* Math.random()
* length
* forEach()
* splice()
* clearInterval()
* setInterval()
* some()
* innerHTML()
* every()
* add()
* remove()
* contains()
